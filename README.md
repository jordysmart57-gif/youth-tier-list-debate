# Youth Tier List Debate

A polished React + Vite tier-list debate game for youth ministry nights.

The app is built for a youth pastor projecting to a room. Students debate the placement of each item, and the leader controls the final board.

## Features

- Category select, tier board, and final results screens
- Drag-and-drop tier placement
- Click-to-place controls
- Presentation mode
- Edit mode for categories and cards
- Debate timer with 15, 30, and 60 second presets
- LocalStorage persistence
- Keyboard shortcuts

## Run Locally

```bash
npm install
npm run dev
```

Then open the local URL Vite prints in the terminal.
